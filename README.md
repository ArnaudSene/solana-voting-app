# Voting appplication using Solana blockchain

This application is just for testing

## Prequisite

- Rust
- Solana
- Yarn
- Anchor

[Install Solana Anchor](https://www.anchor-lang.com/docs/installation)


## Smart contract

```bash
cd program/vote
anchor build
```


## Front application

```bash
cd app
pnpm i
pnpm run dev
```
